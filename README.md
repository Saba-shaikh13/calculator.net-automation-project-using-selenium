Calculator.net Automation Testing using Selenium WebDriver and TestNG

Developed an automation testing project to validate the functionality of the Calculator.net web application using Selenium WebDriver with Java. The project automates various calculator operations such as addition, subtraction, multiplication, and division to ensure accurate results and proper user interface behavior.

TestNG framework was used to organize test cases, execute test suites, and generate execution reports. Maven was used for dependency management and project build automation.
Technologies Used:
Java, Selenium WebDriver, TestNG, Maven, Eclipse, GitHub
